ArchStudio Mailing Lists

We maintain the following separate mailing lists that are used for communication primarily about ArchStudio but also encompass related projects (xADL, Mighty Automated Software Tools, etc.) We strongly encourage developers working with these tools to subscribe to one or more of these lists.

All our lists are managed by the Mailman program, which allows for convenient Web-based subscription, account management, and archive browsing. Subscribers can receive messages individually or periodically in digest format. All messages from these lists are prefaced in the subject line with the name of the list for easy message filtering and categorization.

archstudio-announce
Main Page / Subscribe Archives Manage Member Options Administration
This is a very low traffic announce-only list that is used only for announcing important ArchStudio-related news such as new builds, new features, or major changes of import to the user community. Only list administrators and moderators can post to this list. We recommend that all ArchStudio users subscribe to this list.

 

archstudio-dev
Main Page / Subscribe Archives Manage Member Options Administration
This is a moderate traffic discussion list that is used for general discussion of ArchStudio-related development issues. Any member can post to this list. We recommend that all ArchStudio developers or those interested in the day-to-day goings on of ArchStudio subscribe to this list.
